Privacy Policy

Effective Date: September 29, 2024

Last Updated: July 5, 2025

Definitions

For the purposes of this Privacy Policy:

  • Bot refers to the Ducky Discord bot and any services provided by it within Discord.
  • Ducky Developers (also referred to as "We," "Us," or "Our") means the individuals responsible for creating, maintaining, and supporting the Ducky bot, including Troptop (@literally.nick) and bobbibones (@bobbibones).
  • Server means a community space within Discord where Ducky operates, containing various channels, members, roles, and configuration settings.
  • Server Data refers to information Ducky collects from a Discord server, such as Server ID, Server Name, Member Count, and Role Count.
  • User Data refers to any information that identifies an individual using Ducky, such as User ID, Username, and data collected during interactions.
  • Interaction refers to an action initiated by a user to perform a function with Ducky, like using a slash or text command, button press, text input, and select menu selection.
  • Command refers to a trigger by the user to the bot, executing a specific function.
  • Audit Logging refers to Ducky's collection of certain server events for record-keeping purposes, including message edits, deletions, role changes, and channel activity.
  • Support Ticket means a request made by a user through the Ducky Support server to obtain assistance, including requests for data access or deletion.
  • You refers to the individual using Ducky, or the server owner or representative who configures Ducky for a server.

Agreement

By using Ducky, you agree to the terms of this Privacy Policy.

Data Processing/Collection

  • Ducky collects data to support the bot's purpose and functionality. Any data provided during an interaction may be processed and stored. Interacting with a server that uses Ducky may result in the processing of that interaction depending on which features are enabled.
  • When a server enables the "Use Bloxlink" feature, Ducky uses Bloxlink's private API (which requires a server key and the user to be a member of the server) to check if your Discord account is linked to a Roblox account.
  • This information is only stored after you confirm it in a direct message (DM) with Ducky.
  • Once confirmed, your Roblox link becomes publicly accessible through Ducky commands or the public API when a valid Discord or Roblox ID is provided. This also applies if you link manually with Ducky through the link command or through the website.
  • If you have questions, please join our support server and create a support ticket.

Access to Data

  • Your data is not shared or sold to third parties.
  • Ducky Developers may access your data for development, debugging, and maintenance purposes. Sensitive credentials such as your PRC API key are never viewed or accessed.
  • Roblox link information is publicly accessible via our API when a valid Discord or Roblox ID is provided, either through a Ducky command or API request.
  • If you would like a copy of your data, please open a support ticket in the Ducky Support server.

Command Usage and Server Activity Logs

For debugging purposes, Ducky logs command usage details, including:

  • Username and User ID of the user executing the command
  • Command name and prefix or slash indicator
  • Timestamp of the command execution
  • Server name where the command was executed

Ducky also logs server activity when it is added to or removed from a server, including:

  • Server logo
  • Server name and Server ID
  • Server owner's name and User ID
  • Server member count at the time of addition or removal

Storage of Data

Your data is securely stored in an SQL database file, accessible only by Ducky Developers, including Troptop (@literally.nick) and bobbibones (@bobbibones).

Server Data is retained for up to 7 days after Ducky is removed from a server. After this period, all associated Server Data is permanently deleted from our systems.

Data Removal Requests

If you would like to request the permanent deletion of your data, please open a support ticket in the Ducky Support server. This action is irreversible.

Requesting data deletion will also remove any associated subscription data for the user. Refunds are not available for deleted subscription data.
