Privacy Policy

Agreement

By using Ducky, you agree to the terms of this Privacy Policy. It is your responsibility to read and stay up-to-date with this document.

Data Collection

Ducky collects data from the server and user through its `/setup` command and any button, select menu, or text input prompt used. The data collected includes, but is not limited to:

  • Server ID
  • Server Name
  • Server Owner & Server Owner ID
  • Server Membercount
  • Server Rolecount
  • Server Channelcount
  • User ID
  • Username

Ducky may also collect additional data provided through server configurations, such as:

  • ER:LC API Key
  • Roles
  • Channels
  • Timestamps
  • In-game server information (via session configurations)
  • Select menu inputs
  • Text input data

Based on server configurations, Ducky may also collect data for audit logging purposes, including:

  • Message edit and deleted content history
  • Role creation/deletion and affected roles
  • Channel creation/deletion and affected channels
  • Reaction add/remove and affected messages/reactions
  • Member join/leave and affected members
  • Built-in Discord Audit Logs

Command Usage and Server Activity Logs

For debugging purposes, Ducky logs command usage details, including:

  • Username and User ID of the user executing the command
  • Command name and prefix or slash indicator
  • Timestamp of the command execution
  • Server name where the command was executed

Storage of Data

Your data is securely stored in a JSON file, accessible only by Ducky Developers, including Troptop (@literally.nick), bobbibones (@bobbibones), and rae (@error40440404040404040404).

Access to your data is limited to debugging purposes, with access only granted with your consent when necessary.

Access to Data

Your data is not shared or sold to third parties. Ducky Developers will not access or use your data without your explicit consent.

If you would like a copy of your data, please open a support ticket in the Ducky Support server.

Data Removal Requests

If you would like to request the permanent deletion of your data, please open a support ticket in the Ducky Support server. This action is irreversible.

Requesting data deletion will also remove any associated subscription data for the user, though this will not affect subscriptions granted to servers by the user. Refunds are not available for deleted subscription data.
