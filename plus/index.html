ERLC Player Panel

Manage a player effortlessly with the ERLC player panel, accessible via /erlc panel <player>.

Ensure users are in the required server before they can enter giveaways. Note: Ducky must be added to the required server.

Export Embed Message Command

Right-click on any message with an embed and export it into a Ducky Exportable.

ERLC Command Queueing

If your server is offline, commands are automatically queued and executed when the server goes online.
