Ducky API

API Documentation

Base URL: https://api.duckybot.xyz

GET /

Get API status and latency information

Response
{
  "code": 200,
  "message": "The Ducky API is online.",
  "data": {
    "latency": {
      "api": {
        "process": number,
        "response": number
      },
      "bot": {
        "websocket": number
      }
    },
    "usage": {
      "memory": number
    },
    "uptime": {
      "epoch": number,
      "readable": string
    }
  }
}
GET /statistics

Get bot statistics

Response
{
  "code": 200,
  "message": "Bot statistics were fetched successfully.",
  "data": {
    "guilds": number,
    "users": number,
    "links": number
  }
}
